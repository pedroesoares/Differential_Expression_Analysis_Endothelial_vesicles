# Proteomic-And-miRNA-Analysis-of-Endothelial-Extracelular-Vesicles

## Abstract
This repository provides the analysis code used to process and integrate multi-omics data from endothelial cell–derived extracellular vesicles under heme- and TNF-α–induced inflammatory conditions.
